# Install

```
npm install
npm install -save @angular/service-worker
```

# http-server

```
npm install -g http-server
http-server -c-1 -P http://localhost:9000 .
```

# Build and Run

```
ng build --prod
npm run server
npm run start:prod
```

# Push Notifications

```
npm install -g web-push
web-push generate-vapid-keys --json

```
